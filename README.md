# Programming_Language_Coursework

## CQL language interpreter in haskell
### By Elliot Alexander and Harry Brown

## Dependencies
* Haskell

  * happy
  * containers
  * missingH

### Installation
sudo apt install haskell-platform </br>
cabal install alex happy containers missingH

## Compilation
ghc Main.hs

## Run
./Main filename.cql
