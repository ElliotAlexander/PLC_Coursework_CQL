# Programming_Language_Coursework

## CQL language interpreter in haskell
### By Elliot Alexander and Harry Brown

## Dependencies
* Haskell
* Cabal
  * alex
  * happy
  * containers
  * missingH

### Installation
```bash
sudo apt install haskell-platform
cabal install alex happy containers missingH
```
## Compilation
```bash
ghc Main.hs
```
## Run
```bash
./Main filename.cql
```
